# Togo Road Accidents Analysis (2022–2024)

A complete Data Science capstone project focused on road accident analysis in Togo using Python, SQL, Machine Learning, and interactive visualization tools.

## Project Objectives

* Analyze accident trends
* Identify high-risk zones
* Build predictive models
* Create interactive dashboards
* Support data-driven road safety decisions

## Technologies Used

* Python
* Pandas
* Scikit-learn
* SQL
* Plotly
* Dash
* Folium
* Machine Learning
